Introducción a JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación de alto nivel utilizado principalmente para crear interactividad en sitios web. Es un componente esencial de la web moderna y se ejecuta en el navegador del usuario.

Configuración Inicial

Para empezar, solo necesitas un navegador web y un editor de texto. Puedes escribir código JavaScript directamente en el inspector de tu navegador o crear un archivo HTML que contenga tu código JavaScript.

Dentro de tu archivo HTML, puedes incluir JavaScript utilizando la etiqueta <script> en el <head> o al final del <body>.

<!DOCTYPE html>
    <html>
    <head>
        <title>Mi Página</title>
    </head>
    <body>
        <!-- Contenido de tu página -->
    
    <script>
        // Tu código JavaScript va aquí
    </script>
    </body>
    </html>

Variables y tipos de datos

Las variables en JavaScript son como cajas donde puedes guardar cosas, como números, texto o cualquier tipo de información. Puedes pensar en ellas como etiquetas pegadas en esas cajas para saber qué hay adentro.

var.

Antes solía ser la forma común de declarar variables, pero no se usa mucho en la actualidad. Puede causar problemas de alcance (scope).

var nombreDeVariable= "Texto";
var nombre = "Juan"

Let.

Es la forma moderna de declarar variables que pueden cambiar su valor después de ser declaradas. Es más segura que var.

let nombreDeVariable = contenidoDeLaVariable;
let edad = 30;

const.

Se utiliza para declarar variables que no pueden cambiar su valor una vez asignado. Es útil para valores que no deben cambiar.

const pi = 3.1416;

Tipos de datos en JavaScript.

JavaScript tiene varios tipos de datos que puedes guardar en esas cajas (variables).

Número (Number).

Para guardar valores numéricos, ya sean enteros o decimales.

let edad = 25;
    let precio = 9.99;

Texto (String).

Para almacenar cadenas de caracteres (letras, números, símbolos) dentro de comillas simples o dobles.

let nombre = "Maria";
    let mensaje = 'Hola, mundo!';

Booleano (Boolean).

Para representar valores verdaderos (true) o falsos (false), útil en decisiones lógicas.

let variableVerdadera = true;
    let variableFalsa = false;
  • > Mayor que
  • >= Mayor o igual que
  • < Menor que
  • <= Menor o igual que
  • != Diferente de

Arreglo (Array).

Para almacenar múltiples valores en una sola variable, usando corchetes [].

let frutas = ["manzana", "banano", "naranja"];

Objeto (Object).

Para agrupar datos relacionados utilizando llaves {}.

let persona = {
        nombre: "Juan",
        edad: 30,
        ciudad: "Madrid"
    };

Nulo (Null).

Representa la ausencia intencional de cualquier valor o objeto.

let resultado = null;

Indefinido (Undefined).

Cuando una variable se declara pero no se le asigna ningún valor.

let sinDefinir;

Tipos de operadores logicos y de asignación comparativa.

Los operadores de asignación en JavaScript son herramientas que te permiten asignar valores a variables de una manera más rápida y conveniente. Aquí tienes una explicación sencilla de los operadores de asignación más comunes:

Igual (==).

Compara si dos valores son iguales, sin tener en cuenta el tipo de dato. Si son iguales, devuelve true, de lo contrario, devuelve false.

5 == "5" // true, porque ambos valores son iguales en valor

No Igual (!=).

Compara si dos valores no son iguales. Si son diferentes, devuelve true, de lo contrario, devuelve false.

5 != 3 // true, porque 5 no es igual a 3

Igualdad Estricta (===).

Compara si dos valores son iguales en valor y tipo de dato. Si son iguales en ambos aspectos, devuelve true, de lo contrario, devuelve false.

5 === "5" // false, porque los tipos de datos son diferentes

No Igual Estricta (!==).

Compara si dos valores no son iguales en valor o tipo de dato. Si son diferentes en cualquiera de estos aspectos, devuelve true, de lo contrario, devuelve false.

5 !== "5" // true, porque los tipos de datos son diferentes

Operadores Lógicos:

AND lógico (&&).

Se utiliza para verificar si dos condiciones son verdaderas. Devuelve true si ambas condiciones son verdaderas; de lo contrario, devuelve false.

(5 > 3) && (10 < 20) // true, porque ambas condiciones son verdaderas

OR lógico (||).

Se utiliza para verificar si al menos una de dos condiciones es verdadera. Devuelve true si al menos una condición es verdadera; de lo contrario, devuelve false.

(5 < 3) || (10 < 20) // true, porque la segunda condición es verdadera

NOT lógico (!).

Se utiliza para invertir el valor de una condición. Si la condición es verdadera, devuelve false, y si la condición es falsa, devuelve true.

!(5 > 3) // false, porque la condición original es verdadera

Ciclos en Javascript

Los ciclos en JavaScript son estructuras que te permiten repetir tareas de manera eficiente. Hay dos tipos principales de ciclos: for y while.

Ciclo for.

for (inicialización; condición; incremento/decremento) {
      // Código a repetir
    }

El ciclo for es útil cuando sabes cuántas veces quieres repetir una tarea. Aquí está su estructura básica:

  • Inicialización: Establece una variable de control que se usa para contar las repeticiones.
  • Condición: Define cuándo se detendrá el ciclo. Mientras esta condición sea verdadera, el ciclo continuará.
  • Incremento/Decremento: Modifica la variable de control en cada iteración para acercarse al final.

Ciclo while.

Es útil cuando no sabes cuántas veces se ejecutará una tarea, pero tienes una condición que debe ser verdadera para continuar.

while (condición) {
      // Código a repetir
    }

Condición: Mientras esta condición sea verdadera, el ciclo se ejecutará una y otra vez.

Ciclo do...while.

Este ciclo es similar al while, pero garantiza que el código se ejecute al menos una vez antes de verificar la condición.

do {
      // Código a repetir
    } while (condición);

Funciones en Javascript

Que es una función.

Es un bloque de código que realiza una tarea específica cuando se llama. Las funciones son como pequeñas tareas que puedes reutilizar en tu código.

Para declarar una función, utiliza la palabra clave function, seguida de un nombre para la función y paréntesis (). El código de la función se coloca entre llaves {}.

function nombreDeMiFuncion() {
      // Tareas que realizara la funcion
    }

Llamar una Función.

Para ejecutar o "llamar" una función, simplemente escribe su nombre seguido de paréntesis ().

nombreDeMiFuncion(); // Llama a la función

Funciones con Parámetros.

Puedes pasar datos a una función a través de parámetros. Los parámetros se declaran entre los paréntesis cuando defines la función y se utilizan dentro de la función.

function nombreDeMiFuncion(nombreDeMiParametro) {
      console.log("¡Mi texto, " + nombreDeMiParametro+ "!");
    }
    
    nombreDeMiFuncion("Mi texto"); // Llama a la función con un argumento

Valor de Retorno.

Las funciones pueden devolver un valor usando la palabra clave return. Esto es útil cuando deseas obtener un resultado de una función.

function nombreDeMiFuncion(parametroUno, parametroDos) {
      return IntruccionDada;
    }
    
    let resultado = nombreDeMiFuncion(parametroUno, parametroDos); // Llama a la función y almacena el resultado
    console.log(resultado); // Imprime el resultado en la consola

Arrow Functions (Funciones Flecha).

Las funciones flecha son una forma más concisa de escribir funciones anónimas.

let nombreDeVariable = (nombreDeLaFuncion) => {
      console.log("¡Texto, " + nombreDeLaFuncion + "!");
    };
    
    nombreDeVariable("Texto"); // Llama a la función flecha
